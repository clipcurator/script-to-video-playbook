# Character and Scene Consistency

Consistency is one of the hardest parts of AI-assisted film creation.

## Character consistency

Track:

- face
- hair
- clothing
- age
- posture
- role-specific props

## Scene consistency

Track:

- location layout
- color palette
- lighting
- time of day
- recurring props

## Practical tip

Create a project bible before generating final video assets. The project bible should include character sheets, scene references, style rules, and forbidden variations.
