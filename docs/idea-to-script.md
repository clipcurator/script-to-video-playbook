# Idea to Script

Turning an idea into a script works best when the idea is first converted into a clear concept, then into a world, character system, outline, and episode structure.

## Step 1: Refine the idea

Start with one sentence:

```text
A [character] wants [goal], but [conflict] forces them to [choice].
```

## Step 2: Validate the concept

Evaluate:

- genre clarity
- audience fit
- visual feasibility
- short drama potential
- AI generation feasibility

## Step 3: Build the world

Define:

- time period
- location
- core conflict
- tone
- rules of the story world

## Step 4: Create the characters

For each major character, define:

- identity
- personality
- motivation
- relationship to others
- visual direction

## Step 5: Generate the outline

The outline should include:

- setup
- turn
- escalation
- climax
- ending or cliffhanger

## Step 6: Draft the script

A useful AI script draft should include:

- scene description
- dialogue
- actions
- camera hints
- emotional beat
