# Script to Storyboard

The goal of storyboarding is not to turn every sentence into a shot. It is to decide what must be shown, what can be narrated, and how each scene moves the story forward.

## Step 1: Break the script into scenes

For each scene, extract:

- location
- characters
- emotional beat
- key action
- visual requirement

## Step 2: Identify story-carrying shots

Prioritize shots that show:

- a decision
- a reveal
- a conflict
- a transformation
- a strong emotional shift

## Step 3: Create the shot list

For each shot, define:

- shot type
- subject
- camera angle
- motion
- duration
- visual prompt

## Step 4: Check continuity

Review:

- costume consistency
- props
- location logic
- character position
- time of day
