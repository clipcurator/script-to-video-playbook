# Storyboard to Video

Storyboard-to-video workflows work best when each shot already has a clear purpose, visual prompt, and motion direction.

## Inputs

- storyboard panels
- character prompts
- scene prompts
- camera movement instructions
- target aspect ratio

## Camera movement examples

- pan
- push-in
- pull-out
- tracking
- orbit
- static close-up

## Output checks

Before publishing, review:

- visual style consistency
- character continuity
- motion quality
- subtitle fit
- scene order
- story clarity
